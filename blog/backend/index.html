YT Web實驗室

最近發現一個YT頻道(Web實驗室)，有一系列講解Web後端基礎，覺得很棒~!

04. 網路是如何運作的－從瀏覽器到伺服器(YT教學)

Browser > URL > wifi/LAN > Router > Proxy(做快取) > Routers(中華電信..等) > DNS > Proxy(Load balancing) > Port(埠號) > Web Server

  • 流量大的網站會由 Proxy 做負載平衡(Load balancing)

  • Proxy 用途是作為反向代理、Http Cache、負載平衡器、防火牆，如:Nginx

  • 由Reverse Proxy決定Port

  • 正向代理/反向代理(Forward Proxy/Reverse Proxy)
