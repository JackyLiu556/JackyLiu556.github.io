Hugo 佈版至 GitHub

By Jacky liu 0001-01-01 學習筆記

個人網站發佈簡易紀錄?

  • clone gitHub (jackyliu556.github.io) 到本地端

  • 用Git做版控，工具用SourceTree，可用SSH公鑰，http方式2021/08已改成access token

  • 新增或修改content裡的md檔案

  • cmd輸入hugo server 啟動Hugo伺服器

  • 檢視修改成果(Ex: http://localhost:1313/)

  • cmd輸入hugo(前提已安裝Hugo)，產生相對靜態檔案(public)

  • 複製靜態檔(public)更新到git資料夾，並push至github
