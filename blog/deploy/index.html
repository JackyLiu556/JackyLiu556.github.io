Spring Boot 部屬至tomcat

By Jacky liu 2021-06-29 Spring boot

版本

apache-tomcat-8.5.68 [下載]

openjdk version 11.0.11 (cmd 鍵入java -version 查詢版本)

1. 快速創建Spring Boot 專案

2. 新增入口

@RestController
public class MyController {
	
	@RequestMapping("/index")
	public String index() {
		return "Hello,word";
	}
}

3. pom.xml 設定

  • 增加 <packaging>war</packaging>

  • 修改成

    <artifactId>spring-boot-starter-tomcat</artifactId>

    <scope>provided</scope>[說明]

完整 pom.xml :

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
	xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 https://maven.apache.org/xsd/maven-4.0.0.xsd">
	<modelVersion>4.0.0</modelVersion>
		<groupId>org.springframework.boot</groupId>
		<artifactId>spring-boot-starter-parent</artifactId>
		<version>2.5.2</version>
		<relativePath/> <!-- lookup parent from repository -->
	</parent>
	<groupId>com.example</groupId>
	<artifactId>deployDemo</artifactId>
	<version>0.0.1-SNAPSHOT</version>
	<name>deployDemo</name>
	<description>Demo project for Spring Boot</description>
	<packaging>war</packaging>
	<properties>
		<java.version>11</java.version>
	</properties>
	<dependencies>
		<dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-tomcat</artifactId>
            <scope>provided</scope>
        </dependency>
		<dependency>
			<groupId>org.springframework.boot</groupId>
			<artifactId>spring-boot-starter-test</artifactId>
			<scope>test</scope>
		</dependency>
		
		<dependency>
			<groupId>org.springframework.boot</groupId>
			<artifactId>spring-boot-starter-web</artifactId>
		</dependency>
	</dependencies>

	<build>
		<plugins>
			<plugin>
				<groupId>org.springframework.boot</groupId>
				<artifactId>spring-boot-maven-plugin</artifactId>
			</plugin>
		</plugins>
	</build>

</project>

3. 打包.war，Maven install產生war檔案

4. 佈署到apache tomcat

  • apache-tomcat-8.5.68\webapps 路徑下清空後貼入war檔案

  • apache-tomcat-8.5.68\bin 執行批次檔startup.bat

  • 開啟瀏覽器

http://localhost:8080/deployDemo-0.0.1-SNAPSHOT/index

deployDemo-0.0.1-SNAPSHOT為war名，會在webapp自動解開
