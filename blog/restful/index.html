RESTful 風格API

By Jacky liu 2021-06-25 Java

RESTful API 是什麼?

Representational State Transfer，簡稱REST，它是一種網路架構風格，它並不是一種標準但不是規範，減少與前端溝通的成本。

設計為 REST 的系統就可以稱為 RESTful。

如何設計出RESTful API?

  1. 使用http method表示動作

  1. 使用url路徑描述資源之間的階層關係

  1. response body 返回json或是xml格式
